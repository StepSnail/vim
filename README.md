## vim setup

1. git clone from github
2. PluginInstall
3. run scripts


## Mac Env Setup

``
brew install ack
``

### install ctags
1. Install  the ctags
``
brew install ctags
``
2. Generate the ctags file
``
Ctags -R .
``
